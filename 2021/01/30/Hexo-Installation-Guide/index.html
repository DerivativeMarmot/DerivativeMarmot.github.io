国外安装

通过 npm 安装 hexo，其中 -g 表示 global 意思是 全局安装；hexo-cli 就是 hexo 本体了。

国内安装

因为网速或其他的原因，使用 国外安装 的方法在国内可能会安装失败。

我们先用 npm 安装淘宝的镜像源 cnpm，再用 cnpm 来安装 hexo
